字段类型

数字类型

int 整数
long 长整数
float 单精度
double 双精度

字符串

char 字节 定长0-255长度 qixxxxxx 会自动补齐
varchar 字符串 变长0-65535字节 qi

日期

date 日期 YYYY-MM-DD
time 时间 HH:MM:SS
datetime 年月日时分秒
timestamp 年月日时分秒

sql类型

三种语言

ddl 数据定义语言 create drop
dml 数据操作语言 select insert update delete （增删改查）
dcl 数据控制语言 grant

建表规范

create table ruozedata.rz(
id int(11)  not null auto_increment,
name varchar(255) comment '姓名',
age  int(3) comment '年龄',
create_user varchar(255),
create_time timestamp not null default current_timestamp,
update_user varchar(255),
update_time timestamp not null default current_timestamp on update current_timestamp,
primary key(id)
);

update_time timestamp not null default current_timestamp on update current_timestamp
这条语句（on语法）奠定了数据仓库的基础

表名称、字段名称不要写中文，尽量不要汉语拼音

统一风格

第一个字段必须是自增长字段，设置为主键且无业务意义

一张表只有一个主键id，业务字段需要唯一的话，就使用唯一约束来保证

务必要加上crete_user、create_time、update_user、尤其是update_time

业务字段，注释要加上 comment ‘xxx’

增删改查

增：insert into 表(字段) values();
删：delete from 表 where 条件
改：update 表 set 字段= where 条件
差：select * from 表 where 字段

常见的sql语法

过滤 where

select * from ruozedata.rz  where name='ruoze';
select * from ruozedata.rz  where name='jepson' and age=22;

select * from ruozedata.rz  where age=16 or age=12;
select * from ruozedata.rz  where age in (16,12);
select * from ruozedata.rz a where exists (
select b.id from ruozedata.rz b 
where (age=16 or age=12 ) and a.id=b.id 
);

过滤可以使用 and or in等条件

排序 order by

正序
select * from ruozedata.rz order by age;
select * from ruozedata.rz order by age asc;
倒序
select * from ruozedata.rz order by age desc, name desc;

asc、desc分别为正序、倒序，默认是正序

模糊匹配 like

select * from ruozedata.rz where name like 'j%'; 字母j开头
select * from ruozedata.rz where name like '%n'; 字母n结尾
select * from ruozedata.rz where name like '%o%'; 含有字母o
select * from ruozedata.rz where name like '__p%'; 第三位字母为p

%为任意字段，_ 表示为占位符

合并表 union

create table t1(id int,name varchar(255));
create table t2(id int,name varchar(255));
insert into t1 values(1,'ruoze');
insert into t2 values(1,'ruoze');
insert into t2 values(2,'jepson');

加上all不会过滤掉重复的字段，
select * from t1
union all
select * from t2;

不加all，会过滤重复的字段
select * from t1
union
select * from t2;

分组语法 group by … having

分组/聚合函数
sum
avg
max/min
count

不带分组字段的
select sum(age),avg(age),max(age),min(age),count(1)
from ruozedata.rz;

带分组字段的 
select 
name,sum(age),count(1) 
from ruozedata.rz 
group by name 
having sum(age)>18; 

==> 等价于子查询
select * from
(
	select 
	name,sum(age) as sum_age,count(1) as c
	from ruozedata.rz 
	group by name 
) as t where t.sum_age>18;

注：能用having就用having，实在不行就用子查询
